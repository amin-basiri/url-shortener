# Initialize

## Install packages

```shell
python -m pip install pipenv
```
```shell
pipenv install
```
```shell
pipenv sehll
```

## Prepare Develop Environment

```shell
export FLASK_ENV=development
```
```shell
export FLASK_DEBUG=1
```
```shell
export FLASK_APP=urlshort
```

## Start

```shell
flask run
```
